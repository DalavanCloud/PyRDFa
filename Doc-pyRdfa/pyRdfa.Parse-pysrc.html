 55          @param incoming_state: the inherited state (namespaces, lang, etc)

 69          # Handle the special case for embedded RDF, eg, in SVG1.2.  
 70          # This may add some triples to the target graph that does not originate from RDFa parsing 
 71          # If the function return TRUE, that means that an rdf:RDF has been found. No 
 72          # RDFa parsing should be done on that subtree, so we simply return...
