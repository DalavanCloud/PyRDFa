traverse_tree(node, func)

 

Traverse the whole element tree, and perform the function func on all the elements.

Parameters:
  • node - DOM element node
  • func - function to be called on the node. Input parameter is a DOM Element Node. If the function returns a boolean True, the recursion is stopped.
