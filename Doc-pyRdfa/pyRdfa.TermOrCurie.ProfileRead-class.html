Class ProfileRead

source code

Wrapper around the "recursive" access to profile files. The main job of this class is to retrieve term and prefix definitions as well as a default vocab value, by accessing an RDF file stored in a URI as given by the values of the @profile attribute values. Each TermOrCurie class has one instance of this class.
